# ex09_arrays

## Goal

Learn how to declare an array, access elements by index, and print each element with a loop.

## Initial Problems

Initial idea:

```c
int x[] = [1,2,3,4,5,6,7,8,9]
```

Problems:
- array initialization used `[]` instead of `{}` 
- the statement was missing a semicolon
- the approach also assumed arrays have a `.length` property

Example wrong assumption:

```c
x.length
```

C arrays do not have built-in properties or methods like that.

## Final Code

```c
#include <stdio.h>

int main(void) {
        int numbers[5] = {1, 2, 3, 4, 5};

        for (int i = 0; i < 5; i++) {
                printf("%d\n", numbers[i]);
        }

        return 0;
}
```

## Output

```text
1
2
3
4
5
```

## Lessons Learned

- arrays use curly braces for initialization
- array indexes begin at `0`
- C arrays do not store a `.length` property
- loops are the standard way to iterate through arrays
- `sizeof(array) / sizeof(array[0])` can compute an array element count only while the variable is still an actual array in the current scope
