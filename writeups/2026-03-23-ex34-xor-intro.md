# ex34_xor_intro

## Goal

Understand the basic XOR property needed for the XOR file tool project: applying the same key twice restores the original value.

## Program

```c
#include <stdio.h>

int main(void) {
	int x = 5;
	int key = 9;

	int encrypted = x ^ key;
	int decrypted = encrypted ^ key;
	printf("The picked number is %d\nThe key is %d\nThe encrypted is %d\nThe decrypted is %d\n",
	       x, key, encrypted, decrypted);
	return 0;
}
```

## What It Demonstrated

- `x ^ key` changes the original value
- applying `^ key` again to that result restores the original

That is the important XOR property:

```text
A ^ K ^ K = A
```

## Lessons Learned

- XOR is a bitwise operation in C using `^`
- the same XOR key can both transform and reverse the data
- this is enough to understand the core mechanism of a simple XOR file tool
- this does not make XOR with a fixed key secure cryptography
