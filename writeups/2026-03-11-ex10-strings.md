# ex10_strings

## Goal

Learn how strings work in C, how `%s` prints them, and how individual characters can be accessed by index.

## Method 1: Character-by-Character Loop

```c
#include <stdio.h>

int main(void) {
        char string[] = "Hello World!";

        for (int i = 0; i < 13; i++) {
                printf("%c", string[i]);
        }

        printf("\n");
        return 0;
}
```

What this demonstrates:
- a C string is a character array
- characters can be accessed with `string[index]`

Warning:
- the fixed bound `13` is brittle
- if the string changes, the loop can become incorrect and may read past the array

## Method 2: Using `%s`

```c
#include <stdio.h>

int main(void) {
        char string[] = "Hello World!";
        printf("%s\n", string);
        return 0;
}
```

What this demonstrates:
- `%s` prints characters until it reaches the null terminator `'\0'`

## Lessons Learned

- C strings are character arrays terminated by `'\0'`
- `%s` prints an entire C string
- `%c` prints a single character
- manual string iteration should not rely on hardcoded lengths unless the bound is guaranteed correct
