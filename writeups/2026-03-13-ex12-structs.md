# ex12_structs

## Goal

Learn how to define a `struct`, create one variable of that type, and access its fields.

## Initial Problems

Initial attempt:

```c
#include <stdio.h>

int main(void) {
        struct person;
        char person.name[] = "Matahari Ramadhan";
        int person.age = 26;
        printf("Your name is %s, you are %d year/s old", person.name, person.age);
        return 0;
}
```

Problems:
- `struct person;` did not define any fields
- `person.name` and `person.age` were used like declaration targets, which is invalid
- field definitions belong inside the struct type definition

Second attempt still had two issues:
- the struct definition was missing its trailing semicolon
- the struct initializer incorrectly contained declarations instead of values

## Working Version

```c
#include <stdio.h>

struct Person {
        char name[30];
        int age;
};

int main(void) {
        struct Person person = {"Matahari Ramadhan", 26};
        printf("Your name is %s, you are %d year/s old\n", person.name, person.age);
        return 0;
}
```

## Lessons Learned

- a struct type must be fully defined before its fields can be used
- struct definitions end with a semicolon
- a struct variable can be initialized with values in field order
- struct fields are accessed with the `.` operator
